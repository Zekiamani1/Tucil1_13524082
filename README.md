# N-Queens Solver with Brute Force Algorithm

## Deskripsi Program

Queens is a logic game available on the professional networking site LinkedIn. The goal of the game is to place the queen on a colored square board so that there is only one queen on it
each row, column, and color area. Additionally, a queen cannot be placed next to each other
other queens, including diagonally.\
this program is made to be the solver of this game by applying brute force algorithm


## Requirements

- **Java Development Kit (JDK)** 17 or higher
- **Apache Maven** 3.6 or higher
- **Linux operating system**
